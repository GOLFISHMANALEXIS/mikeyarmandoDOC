Funcionalidades de la Aplicación de Ubicación

La aplicación de ubicación proporciona las siguientes funcionalidades:

  • Medición de Distancia: Utiliza el sensor del celular para medir distancias.
  • Localización Geográfica: Determina la ubicación del dispositivo utilizando herramientas de geolocalización.
  • Interfaz Amigable: Ofrece una interfaz fácil de usar para visualizar datos de ubicación y distancia.
